## Getting started
 
* To Download:

>	Ubuntu, Debian, Mint, others:<br>
	  ```
	 	 sudo apt-get install wget
	  ```

>	Fedora, CentOS:<br>
	```
		dnf install wget
	```<br>
	```
		yum install wget
	```

>	openSUSE:<br>
	```
		zypper install wget
	```

>	after installing:<br>
	```
		wget https://github.com/ysh-rael/input-range/archive/refs/heads/main.zip
	```

* To run the project:
>    [click here](https://ysh-rael.github.io/input-range/)
